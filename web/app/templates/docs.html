Documentations du bot Hourglass
Listes des commandes
Commande Description
!stats [user] Stats de l'utilisateur sur le serveur
!allstats [user] Stats de l'utilisateur sur tous les serveurs
!top Top 10 des utilisateurs en fonction du temps passé sur le serveur
!alltop Top 10 des utilisateurs en fonction du temps passé sur tous les serveurs
!server Stats du serveur
Code source
Le projet Hourglass est entièrement open source. Vous pouvez accéder à son code, proposer des améliorations ou signaler des bugs directement sur le dépôt GitHub.
API
Ce bot propose une API publique, sans clé afin de récupérer des données(stats) via des endpoints. Vous pouvez utiliser l'API en vous aidant de la Documentation.
